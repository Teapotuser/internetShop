Фильтр товаров в галерее товаров:
1. Страница (главная) Index.html
Фильтр по цене товара (движок слайдер). И по коллекции (чекбоксы). Чекбоксы всех коллекций.
Есть кнопки "Применить", "Сбросить" фильтр
2. Страница Collection.html
Фильтр по цене товара (движок слайдер). И по категории (чекбоксы). В коллекции товары -- в фильтре чекбоксы не всех категорий, а только тех, что присутсвуют в данной коллекции.
Есть кнопки "Применить", "Сбросить" фильтр
3. Страница Category.html
Фильтр по цене товара (движок слайдер). И по коллекции (чекбоксы). В категории товары -- в фильтре чекбоксы не всех коллекций, а только тех, что присутсвуют в данной категории.
Есть кнопки "Применить", "Сбросить" фильтр
Отдельной страницы для результатов фильтра нет. Наверно, результаты загружать в index.html

Сортировка (наверно ajax) -- Название A-Z, Название Z-A, Цена с дешевых, Цена с дорогих.

Поиск
Отдельная страница с результатами поиска (еще не сверстана мною.) На ней количество найденных товаров и галерея товаров. Фильтр по цене и коллекции.
Есть выпадающее окошко с результатами быстрого поиска (наверно ajax)
